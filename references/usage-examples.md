## 示例1：供应商合作决策

用户说："两家供应商报价差不多，A质量口碑好但交期不稳定，B交期稳但去年有过一次质量事故，怎么选？"

输出：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
**纲目.skill — 纲举目张**

**一、纲（核心判断）**
问题的本质不是选供应商，是**赌你要的是确定性还是安全感**。A给你安全感（质量好），B给你确定性（交期稳）。

**二、目（三个关键行动）**

| # | 行动 | 说明 |
|---|------|------|
| 1️⃣ | ★ **查B的质量事故原因** | 是系统性缺陷还是偶发？后者可以接受，前者不可。 |
| 2️⃣ | 🔍 **问A为什么交期不稳** | 是产能问题还是管理问题？前者可谈，后者无解。 |
| 3️⃣ | 🛡 **小单试B** | 给B一个小订单，实际验证质量是否改善。 |

**三、排除（不做什么）**
✗ 不要纠结价格 — 你说报价差不多，那价格不是变量
✗ 不要比售后条款 — 这是细节，不是决策关键

## 示例2：项目延期危机

用户说："项目还有两周交付，核心模块还没集成完，团队天天加班但效率越来越低，客户已经打电话来催了。"

输出：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
**纲目.skill — 纲举目张**

**一、纲（核心判断）**
**两周时间不够做"全部"，但够做"够用的"**。问题不是"怎么把全部做完"，而是"怎么定义'够用'的最低标准"。

**二、目（三个关键行动）**

| # | 行动 | 说明 |
|---|------|------|
| 1️⃣ | ★ **今晚拉清单** | 哪些功能是"没有就黄"的，哪些是"后续补"的。 |
| 2️⃣ | 🔍 **跟客户摊牌** | 主动告诉客户：核心功能按时，边缘功能延期两周。客户知道预期后通常不焦虑。 |
| 3️⃣ | 🛡 **停加班3天** | 效率已触底，连休三天恢复状态，比继续拖更有效。 |

**三、排除（不做什么）**
✗ 不要加人 — 两周内新人上不了手，反而拖慢团队
✗ 不要写日报复盘 — 没时间了，现在只做决定不做分析

## 示例3：个人职业困惑

用户说："我在现在公司干了5年，待遇不错但学不到新东西了，想跳槽又不确定外面行情，怎么办？"

输出：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
**纲目.skill — 纲举目张**

**一、纲（核心判断）**
你不是在"跳不跳"之间选，是在**安全感 vs 增长空间**之间选。5年不走说明安全感对你很重要，但你想增长。

**二、目（三个关键行动）**

| # | 行动 | 说明 |
|---|------|------|
| 1️⃣ | ★ **先面试再决定** | 不看行情就做决定是盲人摸象。面3家，拿到offer才有比较基准。 |
| 2️⃣ | 🔍 **问自己：涨幅多少才值得冒险** | 心里定个数，低于这个数不值得动。 |
| 3️⃣ | 🛡 **留下也行，但要换个活法** | 如果真不走，申请换岗/接新项目，告诉老板你要增长。 |

**三、排除（不做什么）**
✗ 不要现在辞职 — 没offer裸辞是把自己的牌全部扔掉
✗ 不要只跟同事聊 — 他们跟你同处境，问他们等于照镜子
